This is a restful api build with Node.js
To start the project, use npm init command to initianize the folder of the project.
This information will store in package.json file.
The addon need install for this API including:
    1. express  npm install --save express //express is a framework for Node JS
    2. nodemon  npm install --save-dev nodemon // nodemon is a addon that automatic restart sever after we change the code.
    3. morgan  npm install --save morgan // a logging package, need import in app.js
    4. bodyParser npm install --save body-parser //bodyparser is a module to parse the incoming request
    5. mongoose  npm install --save mongoose // mongoose is a module help to connect MongoDB
